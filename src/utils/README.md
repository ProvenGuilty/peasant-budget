# Utils

Helper functions and utilities used throughout the app.

## What Goes Here

- **Date helpers** - Format dates, calculate pay periods, etc.
- **Number helpers** - Format currency, calculate totals, etc.
- **Data processing** - Filter, sort, group transactions
- **API calls** - OpenAI integration, external APIs
- **Storage** - LocalStorage helpers, data persistence
- **Validation** - Form validation, data validation

## Examples

- `dateUtils.js` - Pay period calculations, date formatting
- `currencyUtils.js` - Format money, calculate totals
- `transactionUtils.js` - Filter, sort, categorize transactions
- `aiUtils.js` - OpenAI API calls for categorization
- `storageUtils.js` - Save/load from LocalStorage
- `validationUtils.js` - Form validation rules

## Guidelines

- Pure functions (no side effects)
- Well-tested and reusable
- Clear function names
- Good documentation
